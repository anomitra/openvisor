# OpenVisor

Go under the hood of your OpenCode sessions. Zero Telemetry. Full Clarity.

A client-only web app for browsing Opencode session exports — messages, tool calls, reasoning chains, token usage, and cost. Everything stays in your browser. No backend, no tracking, no servers.

## Usage

```bash
npm install
npm run dev
```

Open http://localhost:3000 and drop your `export.json`.
